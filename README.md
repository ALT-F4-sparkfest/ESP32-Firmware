# ESP32-Firmware for Busina

The official ESP32 Firmware for Busina, The official frontend repository for Busina, a real-time intelligent public transportation platform designed to improve the commuting experience through live vehicle tracking, ETA prediction, and operational analytics.

This repository will contain the embedded system behind the telemetry used for the Busina, including the sensor that detects whether the jeepney is moving or not, the MQTT protocol for communication between the web and the ESP32 and the GPS module for telemetry of coordinates.
